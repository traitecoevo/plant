Create fixed light environment

Usage

fixed_environment(canopy_openness, height_max = 150)

Arguments

canopy_openness
Index of canopy openness (on 0,1)
height_max
Maximum height. The default (150) should be big enough for most uses.

Description

Create a light environment where light levels are constant down the canopy.
